Principais Projetos

Front-end Development

Um dos projetos que desenvolvi foi o jogo 'Número Secreto', criado para consolidar meus conhecimentos em HTML, CSS e, principalmente, JavaScript. O objetivo do jogo é desafiar o usuário a adivinhar um número secreto, gerado aleatoriamente entre 1 e 10. Utilizei HTML para estruturar o jogo, CSS para estilizar e criar uma interface atraente, e JavaScript para implementar a lógica, incluindo a geração do número, verificação das tentativas e exibição de mensagens de feedback ao jogador. Esse projeto foi uma oportunidade enriquecedora para praticar lógica de programação e melhorar a interação do usuário com a aplicação.

Front-end Development

Outro projeto que desenvolvi foi o 'Decodificador de Texto', criado para proporcionar entretenimento de forma lúdica e criativa. A aplicação permite codificar e decodificar mensagens, transformando textos comuns em enigmas intrigantes que podem ser revertidos ao original com um toque de magia. Utilizei HTML para estruturar o layout, CSS para criar um design envolvente e JavaScript para implementar as funções de codificação e decodificação. Este projeto foi uma experiência divertida e desafiadora, que me ajudou a aprimorar habilidades técnicas e a explorar a criatividade no desenvolvimento de aplicações interativas.

Formação acadêmica

Minha trajetória acadêmica no ensino médio integrado ao técnico em contabilidade foi marcada por dedicação e aprendizado prático. Desenvolvi em equipe um trabalho de conclusão de curso (TCC) com o tema 'A Consultoria Contábil no Regime Tributário', onde explorei como a escolha correta do regime tributário impacta diretamente na lucratividade empresarial. A pesquisa destacou a importância de uma análise estratégica para maximizar os resultados financeiros.
